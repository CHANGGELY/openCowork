# Contributing to openCowork

First off, thank you for considering contributing to openCowork! It's people like you that make openCowork such a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues.
* Provide a clear and descriptive title.
* Include screenshots or recordings if possible.

### Suggesting Enhancements
* Open an issue with the [Enhancement] tag.
* Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo.
2. Create a new branch (`feat/your-feature` or `fix/your-fix`).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

## Development Setup

See the [README.md](README.md) for instructions on setting up the backend and frontend locally.

## Localization
We aim to support multiple languages. If you want to add a translation, please check `frontend/src/app/page.tsx` for the current dictionary.
